📖 Tentang KIR

KIR adalah tempat bagi siswa untuk belajar bersama dengan mengedepankan sikap ilmiah.

🎯 Misi Kami

Menjadi agen perubahan yang solutif melalui inovasi sains dan teknologi untuk mnyelesaikan masalah nyata di lingkungan sekitar demi mesa depan yang lebih baik.

🌟 Visi Kami

1. Mengembangkan potensi diri di luar kurikulum sekolah.

2. Menciptakan inovasi yang tidak hanya baik di atas kertas, namun juga mudah diterapkan di masyarakat.

3. Membuktikan bahwa anak muda juga dapat ikut serta pada dunia ilmiah dengan cara yang seru dan kreatif.

🤝 Keuntungan Bergabung

  • Mengembangkan kemampuan berpikir kritis dan analitis.
  • Meningkatkan keterampilan meneliti dan menulis ilmiah.
  • Berkesempatan mengikuti lomba dan publikasi karya.
  • Berkolaborasi dan berkontribusi untuk kemajuan sekolah.

💻 Mulai Belajar Programming

Programming adalah seni menulis instruksi untuk komputer. Mari kita mulai dengan dasar-dasarnya!

📚 Apa itu Programming?

Programming adalah proses menulis kode untuk memberitahu komputer apa yang harus dilakukan. Seperti memberi resep pada komputer untuk menyelesaikan tugas tertentu.

1️⃣ HTML

Bahasa markup untuk membuat struktur halaman web. HTML memberikan kerangka dasar untuk konten.

2️⃣ CSS

Bahasa styling untuk menghias halaman web. CSS membuat website Anda terlihat cantik dan menarik.

3️⃣ JavaScript

Bahasa pemrograman untuk membuat interaksi. JavaScript membuat website Anda responsif dan interaktif.

📝 Contoh Kode Lainnya untuk Dicoba:

Contoh 1: Variabel dan String 
let nama = "Siswa KIR";
console.log("Halo, " + nama);
Contoh 2: Matematika Sederhana 
let angka1 = 10;
let angka2 = 5;
console.log("10 + 5 = " + (angka1 + angka2));
console.log("10 - 5 = " + (angka1 - angka2));
console.log("10 * 5 = " + (angka1 * angka2));
Contoh 3: Perulangan (Loop) 
for (let i = 1; i <= 5; i++) {
    console.log("Hitung: " + i);
}
Contoh 4: Kondisi (If-Else) 
let nilai = 85;
if (nilai >= 80) {
    console.log("Nilai Anda: A - Sempurna!");
} else if (nilai >= 70) {
    console.log("Nilai Anda: B - Bagus!");
} else {
    console.log("Nilai Anda: C - Perlu Belajar Lebih");
}
